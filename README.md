# RC Car Project

## Overview

This project involved designing and building a remote-controlled (RC) car as part of a college engineering project. The objective was to develop a functional vehicle capable of wireless control while integrating mechanical, electrical, and embedded systems concepts.

## Features

* Wireless remote control operation
* DC motor drive system
* Steering mechanism using servo motors
* Battery-powered system
* Custom chassis assembly
* Real-time control through a handheld transmitter

## Hardware Components

* Microcontroller (Arduino Uno/Nano)
* DC motors with motor driver module
* Servo motor for steering
* Wireless communication module (Bluetooth/RF)
* Rechargeable battery pack
* RC car chassis and wheels

## Software

The control software was developed in C/C++ using the Arduino IDE. The program handled:

* Motor speed control
* Steering angle adjustment
* Wireless command processing
* Safety and power management functions



## Results

The RC car successfully demonstrated wireless movement, steering control, and reliable operation during testing. The project provided practical experience in embedded systems, electronics integration, programming, and system debugging.

## Skills Developed

* Embedded Programming (C/C++)
* Arduino Development
* Electronics and Circuit Design
* Motor Control Systems
* Wireless Communication
* Mechanical Assembly
* Troubleshooting and Testing
